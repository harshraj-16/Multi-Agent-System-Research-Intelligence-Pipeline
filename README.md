# Multi-Agent AI Research Assistant

An AI-powered research assistant built using LangGraph, LangChain, Mistral AI, and Tavily Search.

## Features

- Multi-agent workflow
- Real-time web search
- AI-powered research
- Intelligent report generation
- Modular architecture

## Tech Stack

- Python
- LangGraph
- LangChain
- Mistral AI
- Tavily Search API

Create a .env file:
MISTRAL_API_KEY=your_key
TAVILY_API_KEY=your_key


Run:
python pipeline.py
